# Pokedex

## Technologies

* Vue.js
* Node.js
* [Plapy Platform](https://plapy.soldai.com/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run for production
```
yarn start
```
